     Rock Paper Scissors Game

A game built with HTML/CSS/JavaScript where you can play against
the computer!

    How to Play

1. Choose your move: Rock ✊, Paper ✋, or Scissors ✌️
2. Computer randomly selects its move
3. Winner is determined based on classic rules:
   · Rock beats Scissors
   · Paper beats Rock
   · Scissors beats Paper
4. First to 5 points wins the game!


       3 Key Lessons Learned
1. Random Number Generation
Implementing computer choice using Math.random() taught me how to
generate random outcomes and map them to game elements.

2.Game Logic Implementation
Creating the win/loss logic with conditional statements helped me
understand how to break down complex rules into clean, readable code.

3. State Management
Tracking scores and game rounds demonstrated the importance
of managing application state and updating the UI based on user
interactions.


        Features
1.Persistent Stats: Tracks lifetime wins, losses, and win rate using localStorage.

2.Responsive UI: Modern, mobile-friendly design with a glassmorphism aesthetic.

3Keyboard Support: Play using R, P, or S keys.

     Tech Stack
HTML5: Semantic structure.

CSS3: Custom properties (variables) and Flexbox layout.

JavaScript (ES6): DOM manipulation and local storage API.

     Usage
1.Clone or download the repository.

2.Ensure index.html, style.css, and script.js are in the same directory.

3.Open index.html in any modern web browser.

First player to 10 points wins the round!
