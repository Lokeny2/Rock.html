Rock Paper Scissors Game

A game built with HTML/CSS/JavaScript where you can play against
the computer!

How to Play

1. Choose your move: Rock ✊, Paper ✋, or Scissors ✌️
2. Computer randomly selects its move
3. Winner is determined based on classic rules:
   · Rock beats Scissors
   · Paper beats Rock
   · Scissors beats Paper
4. First to 5 points wins the game!


3 Key Lessons Learned
1. Random Number Generation
Implementing computer choice using Math.random() taught me how to
generate random outcomes and map them to game elements.

3. Game Logic Implementation
Creating the win/loss logic with conditional statements helped me
understand how to break down complex rules into clean, readable code.

3. State Management
Tracking scores and game rounds demonstrated the importance
of managing application state and updating the UI based on user
interactions.
